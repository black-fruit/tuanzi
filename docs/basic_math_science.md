# 基础数学与科学知识卡

## 数学

### 1. 四则运算
- 加法把两个量合并。
- 减法表示一个量减少另一个量。
- 乘法可以理解为重复加法。
- 除法可以理解为平均分配或求包含关系。

### 2. 分数与小数
- 分数表示整体的一部分，例如 `3/4` 表示把整体分成四份后取三份。
- 小数是十进制表示法的一部分，`0.25` 等于 `1/4`。
- 百分数表示每一百份中的多少份，例如 `25% = 0.25 = 1/4`。

### 3. 代数
- 变量表示未知量或可变化的量。
- 方程表示两个表达式相等，例如 `2x + 3 = 11`。
- 解方程的目标是找到使方程成立的变量值。

### 4. 几何
- 三角形内角和等于 `180°`。
- 正方形四条边相等，四个角都是直角。
- 圆的周长公式是 `C = 2πr`，面积公式是 `A = πr²`。

## 科学

### 1. 经典力学
- 速度描述位置变化的快慢，单位常见为 `m/s`。
- 加速度描述速度变化的快慢。
- 牛顿第二定律为 `F = ma`，表示力等于质量乘以加速度。

### 2. 能量
- 能量不会凭空产生或消失，只会从一种形式转化为另一种形式。
- 动能与物体运动有关，势能与位置或状态有关。

### 3. 热学
- 温度反映物体冷热程度。
- 热量会自发地从高温物体传向低温物体。
- 水在标准大气压下约 `0°C` 结冰，约 `100°C` 沸腾。

### 4. 生物
- 细胞是大多数生物体结构和功能的基本单位。
- 植物通过光合作用把光能转化为化学能。
- 呼吸作用帮助细胞释放可利用的能量。

### 5. 地球与天文
- 地球绕太阳公转一周约为一年。
- 月球绕地球运动，并影响潮汐现象。
- 大气层帮助地球维持适合生命的环境。
